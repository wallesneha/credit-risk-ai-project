# AI-Based Credit Risk Prediction System

## 📌 Overview
This project builds a machine learning model to predict loan default risk using financial data. It also uses Explainable AI (SHAP) to interpret model predictions.

---

## 🚀 Features
- Data preprocessing and cleaning
- Random Forest model for classification
- Model evaluation (accuracy + confusion matrix)
- Feature importance analysis
- Explainable AI using SHAP

---

## 📊 Results
- Accuracy: 0.91
- Strong performance in predicting non-default cases
- Key features: credit_type, interest_rate_spread, income, LTV

---

## 🧠 Explainable AI
SHAP is used to understand how features influence predictions.
