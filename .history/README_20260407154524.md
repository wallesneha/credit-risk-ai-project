# Credit Risk AI Project

## Overview
This project predicts loan default risk using Machine Learning.

## Features
- Data preprocessing
- Model training using Random Forest
- Accuracy evaluation
- Feature importance (Explainable AI)

## Tech Stack
- Python
- Pandas
- Scikit-learn

## Result
The model predicts whether a customer is likely to default on a loan and highlights key influencing factors.
